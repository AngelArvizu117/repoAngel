Practica Git

Comando Descripcion
git init Inicializa la carpeta como un repositorio de GIT, para empezar a gestionar el cambio
git add -A Agrega a Stagging
git commit -m Guarda el nuevo commit bajo un nombre
git status Revisa si hay modificaciones
git log Muestra los commits que se han guardado
git config --global user.name Crea un usuario de tipo global
